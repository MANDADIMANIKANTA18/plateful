# 🍽️ Plateful — Smart Food Waste Redistribution Platform

## 📌 Overview
Plateful is a smart platform designed to reduce food waste by connecting 
food donors (restaurants, hotels, events) with NGOs and individuals in need. 
It ensures surplus food reaches the right people at the right time.

## 🚀 Features
- 🔄 Real-time food donation matching
- 📍 Location-based donor and receiver mapping
- 🔔 Instant notifications for available food
- 📊 Dashboard for tracking donations and impact
- 🔐 Secure authentication for donors and receivers

## 🛠️ Tech Stack
- **Frontend:** React.js, CSS
- **Backend:** Java, Node.js
- **Database:** PostgreSQL
- **DevOps:** Docker, Docker Compose
